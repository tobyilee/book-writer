# 11장. 김치를 다 못 먹었다면 — 활용 요리

너무 시어 못 먹겠다는 김치가, 사실 요리에 가장 좋은 김치다.

생으로 먹기엔 시큼한 묵은지, 즉 신김치는 버릴 김치가 아니다. 완전히 발효된 신김치는 익히면 오히려 깊은 맛이 난다 — 신맛이 가열로 둥글어지고, 푹 익은 김치 특유의 감칠맛이 국물과 기름에 우러난다. 10장에서 "망쳤을 때 돌아오는 길"을 이야기했다면, 이 장은 그 길의 끝이다. 담근 김치가 한 바퀴를 돌아 다시 식탁의 주인공이 된다.

여기 여섯 가지는 모두 브리즈번 마트에서 구하는 재료로 만든다. 돼지고기, 두부, 식빵, 달걀 — 특별할 게 없다. 신김치만 있으면 된다.

## 김치찌개

::: meta

**소요 시간** 약 15분
**난이도** 하
**분량** 2인분

:::

::: ingredients

- 신김치(묵은지) 2컵, 한 입 크기로 썰기
- 삼겹살 또는 돼지목살 150g, 썰기
- 두부 반 모, 도톰하게 썰기
- 대파 1대, 어슷썰기
- 김치 국물 0.5컵 + 물 2컵
- 고춧가루 1큰술, 다진 마늘 0.5큰술

:::

::: steps

1. 냄비에 돼지고기를 넣고 중불에서 겉이 익을 때까지 볶는다. 고기 기름이 나오면 김치 맛이 한결 깊어진다.
2. 썬 신김치를 넣고 5분쯤 함께 볶는다. 김치를 먼저 볶아야 군내가 가시고 단맛이 올라온다.
3. 김치 국물과 물을 붓고 고춧가루·다진 마늘을 넣어 끓인다. 김치 국물을 버리지 말고 다 넣는 것이 감칠맛의 핵심이다.
4. 끓으면 두부를 올리고 5분 더 끓인 뒤, 마지막에 대파를 넣는다.

:::

::: tip

신김치가 시면 시을수록 찌개는 깊어진다. 간이 싱거우면 액젓 약간, 너무 시면 설탕 한 자밤으로 잡는다.

:::

## 김치볶음밥

::: meta

**소요 시간** 약 10분
**난이도** 하
**분량** 1~2인분

:::

::: ingredients

- 찬밥 2공기
- 신김치 1컵, 잘게 썰기
- 식용유 2큰술, 김치 국물 2큰술
- 고춧가루 0.5큰술, 설탕 약간
- 달걀 2개, 김 가루·참기름 (선택)

:::

::: steps

1. 팬에 식용유를 두르고 잘게 썬 김치를 중불에서 3분간 볶는다. 김치를 충분히 볶아야 밥에 맛이 밴다.
2. 고춧가루·설탕·김치 국물을 넣고 한 번 더 볶는다.
3. 찬밥을 넣고 김치와 고루 섞어 가며 볶는다. 갓 지은 밥보다 찬밥이 덜 질어 볶음밥에 맞다.
4. 그릇에 담고 달걀프라이를 올린다. 참기름 몇 방울과 김 가루로 마무리한다.

:::

## 김치전

::: meta

**소요 시간** 약 15분
**난이도** 하
**분량** 지름 20cm 2장

:::

::: ingredients

- 신김치 1.5컵, 잘게 썰기 + 김치 국물 3큰술
- 부침가루 1컵 (또는 밀가루 1컵 + 소금 약간)
- 물 0.5~0.7컵
- 식용유 넉넉히

:::

::: steps

1. 볼에 잘게 썬 김치·부침가루·김치 국물·물을 넣고 섞는다. 반죽은 주르륵 흐르되 묽지 않게 — 되면 물을, 묽으면 가루를 더한다.
2. 팬에 식용유를 넉넉히 두르고 중불로 달군다.
3. 반죽을 얇게 펴 올린다. 얇게 부쳐야 가장자리가 바삭하다.
4. 한 면이 노릇해지면 뒤집어 반대 면도 바삭하게 부친다.

:::

::: warning

뜨거운 기름에 반죽을 부을 때 튈 수 있으니, 반죽을 팬 가까이에서 살살 흘려 넣는다.

:::

## 두부김치

::: meta

**소요 시간** 약 15분
**난이도** 하
**분량** 2인분

:::

::: ingredients

- 두부 1모
- 신김치 1.5컵, 썰기 + 돼지고기 100g (선택)
- 식용유 1큰술, 설탕 0.5큰술, 참기름 약간

:::

::: steps

1. 두부를 도톰하게 썰어 끓는 물에 2~3분 데친 뒤 건진다. 데치면 두부가 따뜻하고 부드러워진다.
2. 팬에 식용유를 두르고 (돼지고기를 먼저 볶은 뒤) 김치를 넣어 중불에서 5분 볶는다. 설탕을 약간 넣어 신맛을 잡는다.
3. 마지막에 참기름을 둘러 향을 낸다.
4. 접시에 데친 두부를 둘러 담고 가운데에 볶은 김치를 올린다.

:::

## 김치만두(또는 만둣국)

::: meta

**소요 시간** 약 40분
**난이도** 중
**분량** 만두 약 20개

:::

::: ingredients

- 신김치 1.5컵, 잘게 다져 물기 꼭 짜기
- 두부 반 모, 으깨 물기 짜기
- 다진 돼지고기 150g, 다진 파·마늘·참기름·소금·후추
- 만두피 약 20장 (마트 냉동 코너)

:::

::: steps

1. 다진 김치는 물기를 꼭 짠다. 물기가 남으면 만두소가 질어 터진다.
2. 김치·으깬 두부·다진 고기·양념을 모두 섞어 소를 만든다.
3. 만두피 가운데에 소를 한 술 올리고, 가장자리에 물을 발라 반으로 접어 꼭 붙인다.
4. 찜기에 10분 찌거나, 멸치·다시마 육수에 넣어 끓이면 만둣국이 된다.

:::

::: tip

한 번에 넉넉히 빚어 냉동해 두면, 신김치가 생길 때마다 만들어 쟁여 둘 수 있다.

:::

## 김치 그릴치즈(김치 토스티)

::: meta

**소요 시간** 약 10분
**난이도** 하
**분량** 1개

:::

::: ingredients

- 식빵 2장
- 신김치 0.3컵, 잘게 썰어 물기 짜기
- 체다 또는 모차렐라 치즈 1~2장
- 버터 약간

:::

::: steps

1. 잘게 썬 김치는 물기를 짠다. 물기가 많으면 빵이 눅눅해진다.
2. 식빵 한 장에 치즈를 깔고 김치를 올린 뒤 다른 식빵을 덮는다.
3. 팬에 버터를 약간 두르고 약불에서 양면을 노릇하게 굽는다. 약불로 천천히 구워야 속 치즈까지 녹는다.

:::

::: tip

브리즈번 마트의 사워도우나 손바닥 빵으로 만들면 한층 그럴듯하다. 매운맛이 부담되면 김치를 한 번 헹궈 물기를 짜고 쓴다. 신김치의 신맛과 녹은 치즈가 의외로 잘 어울려, 김치를 처음 접하는 현지 친구에게 권하기 좋은 한 끼다.

:::

이렇게 담근 김치는 생으로도, 익혀서도 식탁을 한 바퀴 돈다. 담그기에서 시작해 먹기와 살리기까지 — 김치의 순환이 여기서 완성된다. 다음 장에서는 이 여정을 돌아보며 책을 닫는다.
