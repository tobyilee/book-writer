# 9장. 더 많은 김치 — 파김치·백김치·오이소박이·총각김치

아이가 매운 걸 못 먹는다면, 주말에 손님이 온다면, 냉장고에 마침 파 한 단이 시들어간다면 — 그때마다 답이 되는 김치가 다르다. 김치는 한 종류가 아니라, 상황과 계절과 식구에 맞춰 골라 담그는 음식이다. 배추김치 하나를 정석으로 익혔다면, 이제 그 양념과 절임 감각을 변주해 레퍼토리를 넓힐 차례다.

이 장에서 담그는 넷은 성격이 뚜렷하게 다르다. 먼저 어느 것이 언제 담그기 좋은지 한눈에 보자.

| 김치 | 담그기 좋은 시기 | 한 줄 성격 |
|------|----------------|-----------|
| 파김치 | 연중 (파 연중) | 절임 없이 양념만, 하루 발효 — 최단·최이지 |
| 백김치 | 연중 | 안 매움 — 아이·매운맛 약한 사람용 |
| 오이소박이 | 10–3월 봄·여름 (오이 제철) | 아삭, 여름용 |
| 총각김치 | 6–8월 겨울 (무 제철) | 깍두기의 통무 변형 |

파김치와 백김치는 처음부터 끝까지 따라 할 수 있게 풀 레시피로 쓴다. 오이소박이와 총각김치는 앞 장에서 배운 것을 역참조해 다른 점만 짚는 변형 레시피다. 가장 쉬운 파김치부터 가자.

## 파김치 — 절임 없이, 하루면 익는다

파김치가 입문 확장으로 좋은 이유는 **절이는 단계가 아예 없기** 때문이다. 파는 절이지 않고 양념만 발라 하루 두면 알아서 숨이 죽고 익는다. 손이 가장 덜 가는 김치다.

::: meta
**소요 시간** 손질·버무리기 25분 + 발효 상온 1일 → 냉장
**난이도** 하
**분량** 반찬통 하나
**담그기 좋은 시기** 연중 (쪽파·실파는 브리즈번에서 연중 구하기 쉽다)
:::

::: ingredients
재료

- 쪽파 또는 실파 300g (가는 것이 잘 익는다)
- 고춧가루 3큰술
- 멸치·까나리 액젓 3큰술
- 다진 마늘 1큰술
- 다진 생강 1/2작은술
- 설탕 1작은술
- 통깨 1큰술
- 찹쌀풀 2큰술 (선택 — 양념이 파에 잘 붙는다)
:::

::: steps
1. 파는 뿌리를 자르고 누런 잎을 떼어낸 뒤 흐르는 물에 흙을 씻어낸다. 흙이 남으면 군내의 원인이 된다.
2. 씻은 파의 물기를 충분히 턴다. 물기가 많으면 양념이 묽어진다.
3. 먼저 **액젓만 파의 흰 뿌리 쪽에 부어 5분 둔다.** 단단한 흰 대가 먼저 숨이 죽어야 나중에 부러지지 않고 잘 감긴다.
4. 고춧가루·다진 마늘·생강·설탕·통깨(찹쌀풀을 쓰면 함께)를 섞어 양념을 만든다.
5. 숨이 죽은 파에 양념을 골고루 바른다. 장갑을 끼고 파를 한 방향으로 살살 훑듯 묻힌다 — 세게 주무르면 풋내가 난다.
6. 서너 가닥씩 모아 돌돌 말아 용기에 담는다. 용기는 8할만 채우고, 발효 중 가끔 눌러 가스를 뺀다.
:::

::: warning
양념을 바른 직후 한여름(24°C 초과)이라면 상온에 오래 두지 않는다. 반나절만에 시어질 수 있으니 기포가 보이면 바로 냉장한다. 상온 발효는 어떤 계절이든 72시간을 넘기지 않는다.
:::

::: steps
7. 뚜껑을 덮어 상온에 **1일** 둔다(겨울은 1~2일, 여름은 반나절~1일). 살짝 익은 냄새가 나면 냉장으로 옮긴다.
:::

### 완성 확인

파가 처음보다 부드러워지고 양념이 배어 색이 짙어지면 익은 것이다. 갓 담갔을 때의 알싸한 생파 향이 가시고 새콤한 맛이 돌면 먹기 좋다.

### 흔한 실수와 복구

- **파가 풋내가 난다:** 너무 세게 주물렀거나 덜 익은 것이다. 하루 더 익히면 풋내가 가신다.
- **너무 짜다:** 액젓이 많았다. 다음번엔 액젓을 2큰술로 줄인다. 이미 짜다면 밥·두부와 곁들여 먹는다.

### 응용

::: tip
대파만 있다면 어슷하게 썰어 같은 양념에 무친다. 더 빨리 먹을 수 있는 대파겉절이가 된다.
:::

## 백김치 — 맵지 않은, 아이도 먹는 김치

백김치는 고춧가루를 전혀 쓰지 않는다. 매운맛 대신 **사과·무·양파의 자연 단맛**과 맑은 국물로 시원하게 먹는 김치다. 매운 걸 못 먹는 아이나 어른, 손님상에 두루 좋다. 이 책에서 차별 가치가 큰 레시피이기도 하다 — 배추김치를 못 먹던 식구가 김치에 입문하는 다리가 된다.

::: meta
**소요 시간** 절임 4~6시간 + 손질·담그기 40분 + 발효 상온 1~2일 → 냉장
**난이도** 중
**분량** 2L 용기 한 통
**담그기 좋은 시기** 연중 (배추는 마트에서 연중 구하기 쉽고, 겨울 제철에 더 좋다)
:::

::: ingredients
재료

- 알배기배추 1통 (약 1kg)
- 절임용 굵은소금 1/4컵 + 물 4컵
- 무 200g (채 썰기)
- 사과 1/2개 (채 썰기 — 단맛)
- 양파 1/4개 (채 썰기)
- 쪽파 3대
- 다진 마늘 1큰술
- 다진 생강 1/2작은술
- 국물: 물 4컵 + 굵은소금 2큰술 + 설탕 1작은술
- 멸치액젓 1큰술 (선택 — 감칠맛, 빼면 더 담백하다)
:::

::: steps
1. 배추를 길게 4등분한다. 큰 통이면 6등분한다.
2. 물 4컵에 소금 1/4컵을 풀어 절임물을 만들고, 배추를 담가 **4~6시간 절인다**(여름은 3~4시간). 잎이 휘어지면 다 절여진 것이다. (6장 통배추는 6~7시간이지만, 알배기배추는 작아 4~6시간이면 충분하다.)
3. 절인 배추는 흐르는 물에 두세 번 헹궈 짠기를 빼고 물기를 짠다. 헹구지 않으면 너무 짜다.
4. 무·사과·양파·쪽파를 채 썰고 다진 마늘·생강과 섞어 소를 만든다. 고춧가루가 없으니 소가 하얗고 깔끔하다.
5. 배추 잎 사이사이에 소를 조금씩 끼워 넣는다. 많이 넣지 않아도 된다 — 맛은 국물에서 더 난다.
6. 소를 넣은 배추를 용기에 차곡차곡 담는다.
7. 국물(물 4컵 + 소금 2큰술 + 설탕, 원하면 액젓)을 만들어 맛본다. **약간 짜다 싶게** 맞춘 뒤 배추가 잠기도록 붓는다.
8. 용기는 8할만 채우고, 배추가 국물 위로 뜨지 않게 눌러 잠기게 한다.
:::

::: warning
국물 김치이므로 배추가 공기에 닿으면 그 면에 골마지가 핀다. **반드시 국물에 잠기게** 누른다. 여름(24°C 초과)이라면 상온 하루를 넘기지 말고 기포가 보이면 즉시 냉장한다.
:::

::: steps
9. 뚜껑을 덮어 상온에 **1~2일** 둔다(여름은 하루 이내). 국물에 기포가 돌고 살짝 시큼해지면 냉장으로 옮긴다.
:::

### 완성 확인

국물이 맑게 시원하고 배추가 아삭하면 성공이다. 사과·무의 단맛과 약한 신맛이 어우러져, 맵지 않아도 김치 맛이 또렷하다. 냉장에서 2~3일 익히면 더 깊어진다.

### 흔한 실수와 복구

- **국물이 밍밍하다:** 백김치는 매운맛이 없어 간이 약하면 맛이 흐려진다. 발효 전이라면 소금을 1작은술씩 더해 "약간 짜게"를 맞춘다.
- **단맛이 부족하다:** 사과를 더 갈아 넣거나 배를 추가한다. 설탕보다 과일 단맛이 깔끔하다.

### 응용

::: tip
배를 구할 수 있으면 사과 대신 배를 넣는다. 더 맑고 시원한 국물이 된다. 잣을 몇 알 띄우면 손님상용으로 모양도 좋다.
:::

## 오이소박이 — 변형 레시피 (양념은 6장 역참조)

오이소박이는 오이에 칼집을 내 그 사이에 소를 채우는 김치다. 아삭하고 시원해 봄·여름에 잘 맞는다(오이 제철 10–3월). **양념 소는 6장 배추김치 양념을 그대로 쓰되 분량만 줄이면 되니**, 여기서는 오이를 다루는 다른 점만 짚는다.

::: ingredients
재료

- 오이 4개 (단단하고 곧은 것)
- 절임용 굵은소금 2큰술
- 부추 50g (소에 섞는다)
- 배추김치 양념 약 1/2컵 (6장 양념 — 고춧가루·액젓·마늘·생강·설탕)
:::

::: steps
1. 오이는 소금으로 표면을 문질러 씻고 양 끝을 자른 뒤 4~5cm 길이로 토막 낸다.
2. 각 토막을 세워, 위에서 아래로 **십자(+) 칼집**을 낸다. 이때 **밑동은 1cm쯤 남겨 자르지 않는다** — 끝까지 자르면 소가 다 빠진다.
3. 칼집 낸 오이를 소금물(물 2컵 + 소금 2큰술)에 20~30분 절인다. 살짝 휘어지면 된다.
4. 절인 오이를 헹궈 물기를 뺀다.
5. 부추를 잘게 썰어 6장 배추김치 양념과 섞어 소를 만든다.
6. 오이 칼집 사이사이에 소를 꼼꼼히 채운다(장갑 권장).
7. 소 채운 오이를 용기에 담는다. 그릇에 남은 양념에 생수를 한두 큰술 더해 헹구듯 풀어, 자작한 국물을 만들어 붓는다. 용기는 8할만 채운다.
:::

::: warning
여름(24°C 초과)에는 상온에 오래 두지 않는다. 반나절이면 기포가 오를 수 있으니, 기포가 보이면 바로 냉장한다. 어떤 계절이든 상온 발효는 72시간을 넘기지 않는다.
:::

::: steps
8. 상온에 반나절~1일 둔다(여름은 반나절). 기포가 오르면 냉장으로 옮긴다.
:::

오이는 오래 두면 물러지니 **익으면 1주 안에 먹는** 김치다.

::: tip
더 빨리 먹고 싶으면 칼집 대신 어슷썰어 양념에 무치면 오이겉절이가 된다. 그날 바로 먹을 수 있다.
:::

## 총각김치 — 변형 레시피 (7장 깍두기 역참조)

총각김치는 알타리무(총각무)를 무청째 통으로 담그는 김치다. 무를 다루는 원리는 **7장 깍두기와 똑같다** — 소금과 설탕으로 절여 물을 빼고, 고춧가루로 색을 입힌 뒤 나머지 양념을 무친다. 중복되는 절임·양념 과정은 7장을 보고, 여기서는 통무라서 달라지는 점만 짚는다.

::: ingredients
재료

- 알타리무(총각무) 1단 (무청 달린 것)
- 절임용 굵은소금 1/4컵 + 설탕 1큰술
- 양념: 7장 깍두기 양념 한 분량 (고춧가루·액젓·마늘·생강·파·설탕)
:::

::: steps
1. 알타리무는 무청(잎) 사이의 흙을 솔로 꼼꼼히 씻는다 — 통으로 담그니 흙이 남기 쉽다.
2. 잔뿌리를 다듬고, 무가 굵으면 세로로 반 갈라 양념이 배게 한다(가는 것은 통째로).
3. 무에 굵은소금과 설탕을 뿌려 **상온 30분~1시간 절인다.** 깍두기처럼 무청까지 숨이 죽으면 된다(7장 절임 요령 참고).
4. 절인 무를 헹궈 물기를 뺀다.
5. 7장 깍두기 양념을 그대로 만들어, 무에 고춧가루로 먼저 색을 입힌 뒤 나머지 양념을 무청까지 골고루 무친다.
6. 무청을 무 몸통에 돌돌 감아 용기에 담는다. 8할만 채우고 가끔 눌러 가스를 뺀다.
:::

::: warning
무 제철인 6–8월 겨울에 담그면 상온 2~3일 발효가 안정적이다. 여름에 담근다면 상온을 하루로 줄이고 즉시 냉장한다(24°C 초과 시 상온 72시간 금지).
:::

::: steps
7. 상온에 2~3일 두어 초기 발효를 시킨 뒤 냉장으로 옮긴다. 냉장에서 1~2주 익히면 아삭하고 알싸한 총각김치가 된다.
:::

::: tip
무가 매우면 절일 때 설탕을 조금 더 넣는다. 겨울 알타리무는 단단하고 단맛이 좋아 통김치에 잘 어울린다.
:::

네 가지를 더하면 식탁이 한결 넉넉해진다. 다음 장에서는 담근 김치를 어떻게 보관하고, 너무 시어지거나 흰 막이 피거나 군내가 났을 때 어떻게 되살리는지 — 발효를 끝까지 관리하는 법을 다룬다.
