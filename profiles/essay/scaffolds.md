# essay 챕터 스캐폴드

에세이 챕터(한 편의 글) 유형별 구조다. 에세이는 형식을 느슨하게 다루는 장르다 — 스캐폴드는 안전망일 뿐, 글의 흐름이 우선한다.

## 1. 일화-성찰형 (가장 흔함)

```
[일화] 구체적 장면·기억·관찰로 연다
[머무름] 그 장면을 충분히 그린다 (서둘러 교훈으로 넘어가지 않기)
[전환] "그런데 생각해보면", "어쩌면"으로 보편으로 들어 올린다
[사색] 그 경험이 건드리는 더 큰 질문을 펼친다
[회귀] 처음의 구체로 돌아와 닫는다 (수미상관)
```

## 2. 단상 몽타주형

여러 작은 장면·생각을 병치해 하나의 정서를 만든다.

```
[조각 1] 작은 장면/생각
[조각 2] 다른 각도의 장면/생각
[조각 3] ...
[수렴] 흩어진 조각들이 가리키는 하나의 정서·통찰
```
- 조각 사이는 공백(`***` 또는 빈 줄)으로 구분. 억지로 잇지 않는다

## 3. 변증·질문형

하나의 물음을 여러 각도로 돌려본다.

```
[질문] 글 전체를 끌고 갈 하나의 물음
[입장 A] 한쪽으로 생각해본다
[흔들기] "하지만 정말 그럴까"
[입장 B] 반대 결도 들여다본다
[잠정 결론] 답을 단정하지 않되 자기 자리를 정한다
```

## 4. 인물·대상 스케치형

한 사람·사물·장소를 그리며 사유한다.

```
[대상 제시] 누구/무엇을 그릴 것인가
[구체 묘사] 감각적 디테일로 그 대상을 살려낸다
[관계] 그 대상과 나의 관계·기억
[의미] 그 대상이 나에게/우리에게 무엇인가
```

## 5. 계절·시간형 (수필적)

흐르는 시간·계절·일상의 결을 따라간다.

```
[지금] 현재의 한 장면
[흐름] 시간·계절의 변화를 따라 사유가 이동
[축적] 변화 속에서 길어 올린 생각
[여운] 닫지 않고 흐르게 두는 마무리
```

## 공통 가이드

- **구체로 열고**, 추상으로 올라갔다가, 가능하면 구체로 닫는다
- 교훈을 강요하지 않는다. 독자가 자기 생각을 얹을 여백을 남긴다
- 문장 길이를 변주한다. 긴 사색 문장 뒤 짧은 한 문장으로 끊어 강조
- 분량 가이드: 한 편당 2,000~5,000자(한글). 단상형은 더 짧을 수 있다
- 소절(`###`)은 강제하지 않는다. 한 편이 끊김 없이 흐르는 게 자연스럽다면 그대로 둔다
