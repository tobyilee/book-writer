# essay 스타일 체크리스트

style-guardian이 에세이 챕터(한 편) 초안을 검수할 때 쓰는 항목이다. 원문 인용 → 대안 형식. 한 편당 5~10건.

## 에세이 항목

| 항목 | 확인할 것 |
|------|----------|
| 구체로 열기 | 추상론·일반론부터 시작했는가? → 장면·경험으로 여는 제안 |
| 일화 머무름 | 일화를 펴자마자 교훈으로 급히 넘어갔는가? → 충분히 머무르기 |
| 설교조 | "우리는 ~해야 한다"식 교훈 강요인가? → 사색·여지("어쩌면 ~인지도")로 |
| tech-book 잔재 | "찜찜하다·해보자" 같은 개발자 공감어·실습 청유가 섞였는가? → 사색 어조로 |
| 1인칭 균형 | 자기 연민·자기 자랑에 빠졌는가? |
| 미문 과잉 | 수사·미문이 진심을 덮는가? → 절제 |
| 인용 사용 | 인용으로 권위를 빌리는가? → 자기 사유의 거울로 |
| 문장 리듬 | 비슷한 길이 문장만 이어지는가? → 길이 변주, 짧은 강조 |
| 여백 | 결론을 못 박아 독자 생각 여지를 없앴는가? |
| 회귀·마무리 | 흐지부지 끝나는가? → 여운 또는 수미상관 |

## 우선순위 라벨

- **Critical**: 일화 없이 추상론 일색, 노골적 설교, 장르 톤 붕괴(tech-book 어조)
- **Should**: 급한 전환, 미문 과잉, 1인칭 균형 무너짐
- **Nice**: 문장 리듬·단어 선택 윤문
