# practical 스타일 체크리스트

style-guardian이 실용서 챕터 초안을 검수할 때 쓰는 항목이다. 원문 인용 → 대안 형식. 한 챕터당 5~10건.

## 실용서 항목

| 항목 | 확인할 것 |
|------|----------|
| 분량·단위 구체성 | "적당히", "약간" 같은 모호어가 기준 없이 쓰였는가? → 수치·기준 병기 |
| 단계 단일성 | 한 단계에 두 동작이 섞였는가? → 분리 제안 |
| 단계 순서 | 순서가 논리적인가? 빠진 단계는 없는가? |
| 위험 경고 위치 | 위험 경고가 동작 뒤에 있거나 빠졌는가? → 동작 앞으로 |
| 메타 정보 | 시간·난이도·분량/예산 요약 박스가 있는가? |
| 구조화 블록 | 재료·준비물이 산문에 묻혀 있는가? → 목록/체크박스로 |
| 검증 안 된 단정 | "무조건 된다"식 단정인가? → 조건·복구법 병기 |
| 복구·실패 대비 | 망쳤을 때 돌아오는 법이 있는가? |
| 현지·개인차 고지 | 환경 차이(오븐 화력·현지 사정)를 무시한 단정인가? |
| 이유 제시 | 중요한 단계에 "왜"가 한 줄 있는가? (응용·복구 가능하게) |

## 우선순위 라벨

- **Critical**: 안전 위험 누락, 단계 누락으로 따라 하면 실패, 위험한 단정
- **Should**: 모호한 분량, 단계에 두 동작 섞임, 메타 정보·복구법 누락
- **Nice**: 구조화 블록 정리, 이유 한 줄 보강

> 안전 위험(화상·부상·식품 안전 등) 누락은 항상 Critical이다. 실용서는 독자가 실제로 행동에 옮긴다.
