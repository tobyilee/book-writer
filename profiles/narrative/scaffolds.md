# narrative 챕터 스캐폴드

서사물의 챕터/씬 구조다. 기술서처럼 "유형"으로 딱 떨어지지 않는다 — 한 챕터는 보통 하나의 씬 또는 몇 개의 씬 시퀀스로 이뤄진다. 억지로 맞추지 말고 이야기의 자연스러운 호흡을 우선한다.

## 챕터 위의 구조: 3막

전체 책(또는 큰 부)은 3막 골격으로 본다. book-planner가 챕터를 배치할 때 참고한다.

```
[1막 설정] 일상 세계 → 사건 발단(inciting incident) → 1막 전환점(돌이킬 수 없는 선택)
[2막 대립] 상승하는 시련 → 중간점(midpoint, 판이 뒤집힘) → 위기 → 2막 전환점(최저점)
[3막 해결] 절정(climax) → 하강 → 결말(resolution)
```

## 씬 스캐폴드 (챕터의 기본 단위)

모든 씬은 작은 이야기다. 욕망과 장애물이 있어야 한다.

```
[진입] 시간·장소·시점 인물을 빠르게 정착 (감각 디테일 1~2개)
[목표] 시점 인물이 이 장면에서 원하는 것
[갈등] 그것을 막는 것 (인물·상황·내면)
[전환] 상황이 바뀐다 — 작게라도. 들어올 때와 나갈 때가 달라야
[훅] 다음으로 끌고 갈 긴장이나 질문을 남기고 닫는다
```

## 챕터 유형별 변주

### 1. 오프닝 챕터 (1장)
```
[훅] 첫 문장이 의문·긴장·이미지로 독자를 붙든다 (배경 설명으로 시작 금지)
[일상] 주인공의 평소 세계와 결핍을 장면으로 보여주기
[균열] 끝에 변화의 씨앗(사건의 그림자)을 심는다
```

### 2. 발단·전환점 챕터
```
[평온의 끝] 돌이킬 수 없는 사건
[반응] 주인공의 선택 — 성격이 드러나는 순간
[새 국면] 이야기가 새 방향으로
```

### 3. 대립·시련 챕터 (2막 다수)
```
[시도] 주인공이 목표를 향해 움직인다
[역풍] 시도가 새 문제를 부른다 (해결이 곧 다음 위기)
[대가] 잃는 것을 보여준다 — 긴장 누적
```

### 4. 절정 챕터
```
[몰림] 빠져나갈 곳 없는 상황
[대결] 핵심 갈등의 정면 충돌
[변화] 주인공이 (또는 세계가) 돌이킬 수 없이 바뀐다
```

### 5. 결말 챕터
```
[여진] 절정의 결과를 보여준다 (설명 아닌 장면)
[새 일상] 달라진 세계 — 1장의 일상과 대조
[여운] 닫되 다 닫지 않는다. 독자에게 남길 정서
```

## 공통 가이드

- 챕터는 **장면으로 열고 장면으로 닫는다.** 작가의 요약·해설로 시작하지 않는다
- 시점은 챕터(또는 씬) 단위로 고정. 중간 점프 금지
- 연속성을 의식한다: 인물 이름·외모·관계·타임라인·이전 챕터의 복선이 어긋나지 않게. 저술 전 `{slug}/story_bible.md`(정전)를 읽고 캐논에 맞춘다 — `continuity-keeper`가 이 바이블로 모순을 잡는다
- 분량 가이드: 챕터당 4,000~8,000자(한글). 장르·페이싱에 따라 신축
- 대사·행동·묘사의 비율을 의식한다. 설명 문단이 길게 이어지면 장면으로 환원할 수 있는지 점검
