# Toby 문체 확장 가이드 → 이전됨

하네스 v1.3.0부터 voice 가이드는 장르 프로필로 이관되었다. 이 문서의 내용은 **tech-book 프로필**로 옮겨졌다.

→ **`profiles/tech-book/voice.md`** (프로젝트 루트 기준)

챕터를 저술할 때는 오케스트레이터가 확정한 활성 장르(`{slug}/book_manifest.json`의 `genre`)에 해당하는 `profiles/{genre}/voice.md`를 읽는다. 장르가 없으면 기본값 `tech-book`. 프로필 구조는 `profiles/_registry.md` 참조.
