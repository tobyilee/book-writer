- 🕒 **[주석 (a)] "대화 길어지면 마크다운 지시 흐려져 몇 메시지마다 재명시"** (원 주석: GPT-5 가이드 "3~5 메시지마다"를 GPT-5.5로 외삽)

  - **판정·처리:** 🕒(시점/계열 외삽, 단 근거 있음). 본문 산문은 이미 "주기적으로"로 수치를 빼 안전 서술. **직접 정정함** — `(사실 확인 필요 …)` 원시 주석을 제거하고 "GPT-5 계열 공통 가이드는 몇 메시지마다 재명시하라고 권한다 — GPT-5.5에서도 같은 결로 적용된다고 보면 된다"로 계열 귀속을 명기하는 깨끗한 산문으로 해소.
- 🕒 **[주석 (b)] "Responses API / `previous_response_id`로 추론 맥락 보존"의 GPT-5.5 적용**
  - **근거:** web.md §2 — "Responses API + `previous_response_id`: 추론 컨텍스트 보존 → eval 73.9%→78.2% 향상"은 GPT-5 가이드에서 확인. GPT-5.5 전용 동작 차이는 미확인.

- 총평: `(사실 확인 필요)` 주석 2건 모두 레퍼런스 근거로 🕒 판정 후 **계열 귀속 명기 + 원시 주석 제거**로 해소. 미해소 주석 0건. 본문이 GPT-5.5 단정으로 새지 않도록 귀속을 계열 수준으로 약화함.
