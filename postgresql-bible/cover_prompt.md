### 비고
- 첫 시도에서 SVG `linearGradient` fill이 ImageMagick 래스터화 시 검정으로 떨어져 단색 fill로 교체.
- 그레인 텍스처를 over compose로 합성 시 전체가 까맣게 덮이는 버그 발생 → 그레인 단계 제거. 매끈한 마감이 오히려 바이블다운 톤에 더 적합했음.
- 폴백 ImageMagick 경로로 외부 API 없이도 충분한 품질 확보. 추후 더 정교한 일러스트가 필요하면 gpt-image-1 등 외부 모델로 재생성 가능.
