# ch02-std-matrix — 같은 코드, 다섯 표준

`std_demo.c`는 C89부터 C23까지 다섯 표준 모두에서 빌드된다. `make matrix` 한 줄이면 다섯 번 빌드하고 다섯 번 실행해서, 컴파일러가 알려주는 `__STDC_VERSION__` 값이 표준마다 어떻게 바뀌는지 한눈에 보여준다. `c23_only.c`는 반대 방향이다 — `nullptr`과 `_BitInt(8)`을 쓰기 때문에 C23에서만 컴파일된다. `make c23-only`로 성공을 보고, `make c23-only-as-c17`로 실패의 모습까지 같이 보자. 표준이라는 게 추상이 아니라는 걸 손으로 느끼는 게 이 디렉토리의 목표다.
