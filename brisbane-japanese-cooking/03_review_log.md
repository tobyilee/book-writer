## 리뷰 라운드 1

레퍼런스 대조: §1~§7 자료가 챕터에 거의 빠짐없이 매핑됨(이론4·구득2·도구3·안전·카테고리·비교 전장·링크). 커버리지 정합성은 합격선. 약점은 "종합 레퍼런스" 표방 대비 식문화 폭과 도구–레시피 불일치.

| # | 라벨 | 축 | 지적 | 근거 | 반영 |
|---|------|----|------|------|------|
| 1 | Critical | 중복·공백 / 실용서 특수 | 다마고야키 팬을 사라면서 다시마키 타마고 레시피가 어느 장에도 없음 — 산 도구를 쓸 레시피 부재 | 레퍼런스 §3(팬=다시마키용) + 책 콘셉트(도구로 맛 경험) | ✅ 반영 — 7장 대표 레시피로 추가(다시·사시스세소·다마고야키 팬 통합 레시피), 말기 연속 사진 지점 명시 |
| 2 | Should | 챕터 균형 | 8장(스시 개론, 9,000자, 레시피 없음)이 독립 챕터로 빈약 | 12장 라멘(16,000)의 56% | ✅ 반영 — 구 8장(개론)+구 9장(안전)을 한 챕터(현 8장 "알아야 안전하게 시작한다", 13,000자)로 통합. 한 장 안에서 "안전→실전" 게이트 순서 유지 |
| 3 | Should | 커버리지 / 누락 | 츠케모노가 7장 "입문" 한 줄, 차·음료 전무 | 리뷰 기준 명시 누락 주제, 이치주산사이 고정 구성요소, 한식 김치 비교 강력 | ✅ 반영 — 츠케모노(아사즈케·누카즈케)를 7장 독립 절로 승격+김치/겉절이 비교 박스. 차·음료(녹차·호지차·맥차) 절을 한 상 차리기 장에 추가 |
| 4 | Should | 커버리지 | 계절감 한 줄 처리 + 퀸즐랜드 제철 공백 | 레퍼런스 §9(6) 후속 항목, "현지 재료로 일식" 핵심 차별점 | ✅ 반영 — 한 상 장에 "브리즈번의 사계와 일식" 절 신설 + 제철 구체 목록은 Phase 4 리서치 보강 플래그 명시(미해결 3번 연결) |
| 5 | Nice | 실용서 특수 | 참고 이미지 배치가 "집필 시 URL 지정"으로 위임 → Phase 4 리스크 | 시각 의존 동작(마키 말기·덴푸라·다시마키)의 재현성 | ✅ 반영 — 시각 의존 높은 장(7장 다시마키, 9장 마키, 8장 라벨 등)에 단계/완성 사진 지점을 계획 단계에서 명시 |
| 6 | Nice | 커버리지 / 식문화 | 식탁 매너(젓가락·그릇 드는 법·이타다키마스) 전무 | practical 종합 레퍼런스의 식문화 폭 | ✅ 반영 — 한 상 차리기 장에 매너 절 한 단락 추가 |

전체 평: 4부 구조와 안전 게이트의 사시미 실전 전 배치는 옳다. 한식 비교 박스·구조화 블록 규약도 견고. Critical 1건만 필수, 나머지는 분량 예산 내 우선순위로 판단 요청.

## 리뷰 라운드 2

갱신본에서 상단 "반영 요약" 헤더는 6건 반영을 약속했으나 **챕터 본문·아크·분량합계·미해결목록이 옛 16장 구조 그대로** — 편집이 절반만 적용된 상태(헤더 vs 본문 모순, 15장 vs 16장 숫자 충돌). 이를 Critical로 지적.

| # | 라벨 | 지적 | 반영 |
|---|------|------|------|
| 1 | Critical | 헤더 약속(통합·번호당김·다시마키·츠케모노·음료·계절·매너)이 본문에 미반영. 숫자만 15장으로 바꿔 16장 본문과 충돌 | ✅ 반영 — 본문 챕터 블록을 실제로 통합·재작성. 다시마키·츠케모노·음료·계절·매너 절이 본문에 실제 기술됨 |
| 2 | Critical | 통합 후 번호가 본문에서 어긋남(12장→14장 점프, 13장 누락, 꼬리 섹션 stale 번호) | ✅ 반영 — 1~15장 연속 번호로 정정. 분량 합계(장별 itemize)·자기점검(안전 8장 게이트, 실전 9장, §5 5~13장)·미해결목록(어물전 8장, 제철 15장) 모두 통합 후 번호로 일치 |
| 3 | Critical(잔여) | 4부 내러티브 아크 문단(라인 48)이 14·15장 순서를 본문 헤딩과 반대로 서술하여 충돌 | ✅ 반영 — **저자가 아크 문단 대신 챕터 본문을 스왑**해 일치시킴(저자 재량 존중). 최종: **14장 한 상 차리기, 15장 자주 하는 실수(책을 닫는 안전망)**. 아크·헤딩·장 배치 메모·분량 합계·미해결 목록 전부 이 순서로 통일됨 |

**저자 결정 존중 사항**: 라운드 2 잔여 항목에서 리뷰어는 "14=실수, 15=한 상" 순서를 제안했으나, 저자는 반대로 "14=한 상(4부 도착점), 15=실수(마지막 안전망)"로 정했다. 근거("앞 장 레시피를 만들어 본 뒤 실수 장을 읽어야 손에 잡힌다")가 설득력 있고 두 순서 모두 타당하므로 저자 결정을 수용. 평행선 아님 — 정합성만 충족되면 어느 순서든 합의 대상이었음.

전체 평: 라운드 1·2의 실질 제안이 모두 본문에 반영됨. 남은 것은 기계적 번호·문장 정합성이었고 추가 왕복 없이 저자 정정으로 마무리. 도구–레시피 정합(다마고야키→다시마키)과 안전 게이트의 한 장 내 통합이 특히 깔끔.

## 최종 합의 계획 요약 (2회 왕복 종료)

**제목**: 「다시부터 사시미까지: 한식 하던 손을 위한 브리즈번 일식 수업」 (슬러그 `brisbane-japanese-cooking`)
**분량**: 15개 본문 챕터 + 부록, 약 200,000자 / 250p
**구조**: 4부 — 준비(1~4장) → 기초(5~7장) → 카테고리 확장(8~13장) → 통합/마무리(14~15장)

| 부 | 장 | 제목 요지 |
|----|----|----------|
| 1부 준비 | 1 | 한식 하던 손에게, 일식은 가깝고도 멀다 (오프닝·지도) |
| | 2 | 브리즈번에서 일식 재료 구하기 (매장·쌀·대체재) |
| | 3 | 일식 조리도구 — 사서 어떻게 쓰는가 |
| | 4 | 일식의 뼈대: 다시·우마미·사시스세소·기본 썰기 (이론) |
| 2부 기초 | 5 | 다시 뽑기 — 일식 맛의 90% |
| | 6 | 밥 — 단립종 짓기와 오니기리 |
| | 7 | 미소국·**다시마키 타마고**·**츠케모노** — 다시를 펼치다 |
| 3부 본체 | 8 | 스시·사시미: 알아야 안전하게 시작한다 ★ (개론 + 식품안전 게이트) |
| | 9 | 스시 밥·니기리·마키 만들기 (실전) |
| | 10 | 면 입문: 우동과 소바 |
| | 11 | 라멘 — 다섯 요소를 조립하다 |
| | 12 | 구이와 조림: 데리야키·야키토리·니쿠자가 |
| | 13 | 튀김: 덴푸라와 돈카츠 |
| 4부 통합 | 14 | 일식 한 상 차리기 — 차림·**음료**·**계절**·**매너** (4부 도착점) |
| | 15 | 자주 하는 실수와 되돌리는 법 (책을 닫는 마지막 안전망) |
| 부록 | A~D | 구득처·대체재·한일 용어사전·참고 이미지 링크 |

**핵심 설계 보존 사항**:
- 식품안전 게이트(8장)가 사시미 실전(9장) **앞** — 한 장 안에서 "안전→실전" 강제
- 매 요리 챕터 "한식 하던 손에게" 비교 박스(`::: tip`) 고정
- practical 스캐폴드 구조화 블록(`::: meta`/`ingredients`/`steps`/`warning`) 규약 준수
- 도구–레시피 정합: 3장에서 산 다마고야키 팬 → 7장 다시마키 타마고로 활용
- 시각 의존 동작에 사진 배치 지점 사전 명시(7장 말기·9장 마키·8장 라벨 등)

**Phase 4 / 리서치 보강 권장(미해결)**:
1. 브리즈번 로컬 어물전 sashimi-grade 취급 여부 (8장·부록A)
2. 교민 커뮤니티 1차 후기 (도입부 현장감)
3. 퀸즐랜드 제철 식재료의 일식 응용 (14장 "브리즈번의 사계" 절)
4. 가격 정보 — 본문에서 단정 금지, "매장 확인" 표현 유지

**평행선 사항**: 없음. 모든 지적이 저자에 의해 수용·반영되어 합의 종료.
