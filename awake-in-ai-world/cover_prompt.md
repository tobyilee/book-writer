- 팔레트: 크림(#fbf3e4) → 따뜻한 복숭아(#f3d2a6) 세로 그라데이션 = 새벽하늘.

## 사용 도구
- **ImageMagick 7.1.2** (이미지 생성 MCP·외부 API 미연결 → 스킬 폴백 경로). 단색 플레이스홀더가 아니라 다층 합성으로 디자인 표지 제작.
- 한글 폰트: `/System/Library/Fonts/AppleSDGothicNeo.ttc` (직접 경로 지정). 제목은 `caption:`으로 자동 줄바꿈 렌더 후 합성.

## 핵심 명령 요약
```bash
FONT="/System/Library/Fonts/AppleSDGothicNeo.ttc"
# 새벽하늘 그라데이션
magick -size 1600x2560 gradient:'#fbf3e4-#f3d2a6' base.png
# 떠오르는 해 (radial glow를 screen 합성)
magick -size 1100x1100 radial-gradient:'#fff3cf-#f3d2a600' sun.png
magick base.png sun.png -gravity North -geometry +0+1250 -compose screen -composite s2.png
# 지평선 + 빛점 + 연도/화살표(draw) ...
# 제목/부제는 caption:으로 줄바꿈 렌더 후 -composite, 저자 -annotate
magick ... -depth 8 -colorspace sRGB -strip cover.png   # 1600x2560
```
