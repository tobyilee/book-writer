## 점검 항목

- [x] **주제 정합성**: 13장이 모두 "AI 코딩과 천연발효 빵"이라는 두 축을 같이 다루는가? — 모든 챕터에서 두 평면이 교차한다.
- [x] **무라카미 톤**: 메타 선언("이번 장에서는") 없이 정물·시각·고유명사로 시작하는 구조인가? — 1장(2시 17분), 2장(새벽 5시) 등 시간 표지로 연다.
- [x] **Toby 평어체**: 평어체+청유형+수사적 질문이 골고루 들어가게 설계됐는가? — 챕터 핵심 질문 자체가 수사적 질문이다.
- [x] **분량**: 13편 × 평균 2,000자 ≈ 26,000자 + 부속 자료. 무라카미 에세이집 분량과 비슷.
- [x] **내러티브 아크**: 발견 → 리듬 → 위임 → 정체성. 자연스러운 흐름.
- [x] **리서치 활용**: Karpathy, METR, Mark, Sennett, Crawford, Newport, 무라카미, 베이킹 디테일이 챕터마다 적절히 분산 배치됐다.
- [x] **균형**: 베이킹만 다루는 챕터 없음, 코딩만 다루는 챕터 없음. 늘 두 평면이 함께 나타난다.

## 보강 결정

- 챕터 1과 2의 시작이 너무 비슷하지 않게(둘 다 "특정 시각") 1장은 오후, 2장은 새벽으로 분기한다.
- 챕터 7은 학술 인용(METR)이 자칫 무거워질 수 있으므로 본문에서는 비율을 1:5(인용 1, 서술 5) 이하로 유지한다.
- 챕터 13은 클로징인 만큼 가장 짧고 정밀하게(약 1,500자). 마지막 장면은 "식탁 위 빵 + 닫힌 노트북" 한 컷으로.
