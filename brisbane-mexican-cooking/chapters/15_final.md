# 15장. 부록 — 조달처·명칭·대체재·도구·레퍼런스 총정리

요리하다가 막히는 순간은 대개 두 가지다. 재료 이름을 못 알아들어서, 또는 그 재료를 어디서 사야 할지 몰라서. 이 부록은 그 두 순간에 바로 펼쳐볼 수 있게 만든 참조 허브다.

서술과 조달 이야기는 2장에서 이미 풀었다. 여기서는 표로 찾는다. 재료 이름이 생각나지 않으면 명칭 대조표(§2), 어디서 사야 할지 모르면 조달처 디렉터리(§1), 재료가 없으면 대체재 표(§3), 고추 종류가 헷갈리면 §4, 단위·온도가 낯설면 §5, 더 배우고 싶으면 §6. 필요한 곳을 바로 펼친다.

## §2. 호주↔멕시코↔한국 식품 명칭 대조표

호주 슈퍼마켓 표지판과 멕시칸 레시피, 한국식 이름이 제각각이다. 장을 볼 때 이 표를 펼쳐두면 헷갈리지 않는다.

::: warning
capsicum·eschalot·shallot 항목의 호주 명칭은 검색 기반 부분 확인이다. 현지 슈퍼마켓 표지판을 직접 1차 확인하는 것을 권장한다.
:::

| 호주 슈퍼마켓 명칭 | 멕시코/미국 레시피 명칭 | 한국어 | 비고 |
|------------------|----------------------|--------|------|

| **Capsicum** | Bell pepper | 피망·파프리카 | 빨강·노랑·초록 |
| **Chilli / Long chilli** | Chile fresco | 생고추 | 종류·매운맛 다양 |
| **Jalapeño** | Jalapeño | 할라피뇨 | 일부 매장 취급 |
| **Eschalot** | Shallot | 샬롯 | 작고 갈색 껍질, 양파 계열 |
| **Shallot / Spring onion** | Green onion / Scallion | 쪽파·파 | ⚠️ 호주에서 "shallot"이 쪽파를 뜻하기도 함 — 맥락 확인 |
| **Avocado** | Aguacate | 아보카도 | — |
| **Lime** | Lima | 라임 | — |

## §3. 재료 대체재 종합표

정통 재료를 못 구했을 때 바로 찾는 표다. 대체재는 "비슷하게 흉내 낼 수 있다"는 의미지, 동일하다는 뜻이 아니다. 가능하면 정통 재료를 쓰는 편이 낫다.

| 정통 재료 | 호주 조달 현실 | 호주 대체재 | 대체 시 주의 |
|-----------|--------------|------------|-------------|
| **마사 하리나** | 슈퍼엔 드묾, 온라인/전문점 | Maseca(Chile Mojo·El Cielo), Bob's Red Mill(유기농 매장) | P.A.N.·콘밀·콘플라워는 대체 불가 |
| **토마티요** | 신선품 드묾, 통조림 일부 | 그린 토마토 + 라임즙으로 산미 보강 / 익힌 그린 피망 | 산미가 약하면 라임을 더 넣는다 |
| **에파소테** | 거의 없음 | 멕시칸 오레가노(1:1) / 고수 / 페넬 약간 | 에파소테 특유의 쓴맛은 완전히 재현 불가 |
| **queso fresco** | 라틴 델리·온라인 일부 | 파니르(가장 근접) / 마일드 페타(프랑스·덴마크식) | 숙성 페타는 짠맛이 강하니 소금 조절 |
| **cotija** | 드묾 | 숙성 페타 / 파르미지아노 레지아노 (소량) | 파르미지아노는 짠맛 강함 — 반만 쓴다 |
| **멕시칸 크림(crema)** | 거의 없음 | 사워크림 + 우유(1큰술)로 농도 조절 / 크렘 프레슈 | 사워크림 단독은 너무 뻑뻑할 수 있다 |
| **신 오렌지(seville orange)** | 드묾 | 라임즙 2 : 일반 오렌지즙 1 비율로 산미·당도 혼합 | 완전히 같지 않지만 코치니타 피빌에 충분히 통한다 |
| **앙초 고추(ancho)** | 전문점/온라인 | Chile Mojo·El Cielo·LatinShop | 구하면 꼭 앙초으로 — 한국 건고추와 풍미 다름 |
| **구아히요 고추(guajillo)** | 전문점/온라인 | 위와 동일 | 포솔레·비리아의 색·깊이 핵심, 대체재 없음 |
| **아치오테 페이스트(achiote)** | 아시안/라틴 전문점·온라인 | 파프리카 + 강황 + 오레가노(색·향 근사치) | 아나토(annatto) 씨앗을 구하면 직접 갈아도 된다 |
| **옥수수 껍질(corn husks, 타말용)** | 전문점·온라인 | 바나나잎(아시안 그로서리에서 냉동 취급) | 바나나잎은 유카탄식 스타일로 맛이 달라진다 |
| **멕시칸 오레가노** | 전문점·온라인 일부 | 일반(지중해식) 오레가노 (풍미 다소 다름) | 멕시칸 오레가노는 레몬·시트러스 향이 더 강하다 |
| **호미니(hominy, 포솔레용)** | 드묾 | 통조림 grits / 마른 포솔레 옥수수(온라인) | 포솔레 특유의 식감은 호미니가 아니면 어렵다 |

## §4. 말린 고추 가이드 표

말린 고추는 멕시칸 요리의 색·향·깊이를 결정한다. 종류마다 쓰임새가 다르고, 여러 종류를 섞어 쓰는 것이 보통이다. 표로 익혀두면 레시피를 읽을 때 헷갈리지 않는다.

| 고추 이름 | 원물 (생) | 외형 | 매운맛 | 풍미 특징 | 대표 용도 |
|-----------|----------|------|--------|-----------|-----------|
| **앙초 (Ancho)** | 포블라노 | 넓고 납작, 짙은 갈색·검정 | 낮음 (1,000~2,000 SHU) | 건포도·자두·초콜릿 같은 단맛과 흙내 | 몰레, 고기 마리네이드, 스튜, 카르니타스 소스 |
| **구아히요 (Guajillo)** | 미라솔 | 길고 좁음, 밝은 붉은색 | 중간 (2,500~5,000 SHU) | 밝은 과일·베리, 약간의 산미 | 포솔레 로호, 비리아, 타말 소스, 살사 로하 |
| **파시야 (Pasilla)** | 칠라카 | 가늘고 길며 주름짐, 거의 검정 | 중간 (1,000~2,500 SHU) | 건포도·자두·허브 복합, 앙초보다 진함 | 몰레, 소스, 수프 베이스 |
| **치포틀레 (Chipotle)** | 할라피뇨(훈제 건조) | 주름진 갈색, 훈제 향 강함 | 높음 (5,000~8,000 SHU) | 강한 훈연향·스모키 | 아도보 통조림, 스튜, 비리아, 살사, 마리네이드 |
| **물라토 (Mulato)** | 포블라노 계열 | 앙초보다 더 검정 | 낮음~중간 | 초콜릿·감초 풍미 강함 | 몰레 포블라노(앙초와 혼합) |
| **아르볼 (Árbol)** | 소형 생고추 | 가늘고 짧음, 밝은 붉은색 | 높음 (15,000~30,000 SHU) | 견과·토스티 풍미 + 강한 매운맛 | 살사 마차, 기름 살사, 매운맛 보강용 |

::: tip
**토스팅이 먼저다.** 말린 고추는 기름 없는 코말(또는 무쇠팬)에 30초~1분 눌러 구우면 풍미가 한층 깊어진다. 탄 냄새가 나면 쓴맛이 강해지니 갈색 연기가 나기 전에 꺼낸다. 토스팅 후 뜨거운 물에 20~30분 불려 씨와 꼭지를 제거한 뒤 블렌더에 간다 — 3장 기초 기법을 참조.
:::

::: tip
**앙초+구아히요** 조합은 멕시칸 레드 소스(비리아·포솔레·타말 소스)의 기본 베이스다. 처음 말린 고추를 사야 한다면 이 두 종류부터 시작한다.
:::

## §5. 계량·온도 환산표

### 5-A. 호주 계량 단위 기준

호주 레시피는 미국식 컵/스푼과 비슷하지만, 표준 계량컵 기준이 약간 다를 수 있다. 멕시칸 레시피 대부분은 미국 출처이므로 아래 환산을 참고한다.

| 단위 | 호주 표준 | 미국 표준 | 비고 |
|------|----------|----------|------|
| 1컵 (cup) | 250 ml | 237 ml | 실제 차이 작음 — 레시피 실패의 주원인 아님 |
| 1큰술 (tablespoon) | 20 ml | 15 ml | ⚠️ 호주 1큰술 = 미국 1큰술 + 1작은술. 차이 주의 |
| 1작은술 (teaspoon) | 5 ml | 5 ml | 동일 |
| 1온스 (oz, 무게) | 약 28 g | 28.35 g | — |
| 1파운드 (lb) | 약 450 g | 453 g | — |

::: warning
**1큰술 차이가 결과를 바꾼다.** 특히 소금·향신료·고춧가루는 미국 레시피에서 "1 tablespoon"이라고 하면 15 ml 기준이다. 호주 20 ml 스푼으로 그대로 따르면 33% 더 많이 들어간다. 처음엔 미국 기준(15 ml)으로 측정하고, 간을 보며 조절하는 편이 낫다.
:::

### 5-B. 오븐 온도 환산

| 섭씨 (°C) | 화씨 (°F) | 가스 마크 | 용도 |
|----------|----------|----------|------|
| 150°C | 300°F | 2 | 저온 장시간 조리 (카르니타스·코치니타 피빌) |
| 160°C | 325°F | 3 | 타말·스튜 오븐 마무리 |
| 180°C | 350°F | 4 | 일반 굽기 (엔칠라다·케사디야 오븐 버전) |
| 200°C | 400°F | 6 | 고온 구이 (생선 구이·비리아 타코 마무리) |
| 220°C | 425°F | 7 | 겉 바삭하게 마무리 (코치니타 피빌 뚜껑 열고 마지막 10분) |

::: tip
오븐마다 실제 화력이 다르다. 온도계 없이 쓴다면 첫 조리 시 10분 일찍 확인하는 습관을 들인다. 특히 타말을 찌는 경우는 오븐이 아닌 스토브탑 찜기를 쓰는 편이 안정적이다.
:::

### 5-C. 자주 쓰는 무게 환산

| 재료 | 표준 계량 | 그램(g) 환산 |

## §7. 용어집

| 용어 | 풀이 |
|------|------|
| **닉스타말화 (Nixtamalization)** | 마른 옥수수를 알칼리 용액(석회수)에 끓이고 불려 껍질을 벗기는 전통 공정. 영양 흡수율을 높이고 펠라그라(니아신 결핍)를 막는다. 마사·토르티야·타말의 토대. |
| **마사 (Masa)** | 닉스타말화한 옥수수를 갈아 만든 반죽. 토르티야·소페·타말 모두 여기서 시작한다. |
| **마사 하리나 (Masa harina)** | 마사를 건조·분쇄한 가루. 물을 더하면 마사 반죽으로 복원된다. 콘밀·콘플라워와 다르다. |
| **살사 (Salsa)** | 생 또는 익힌 소스. 피코 데 가요(생살사), 살사 로하(익힌 붉은 살사), 살사 베르데(토마티요 초록 살사) 등. |
| **몰레 (Mole)** | 고추·향신료·견과류·씨앗·초콜릿 등을 갈아 만든 복합 소스. 수십 가지 재료가 들어가기도 한다. |
| **타코 (Taco)** | 토르티야에 속(필링)을 채운 형태. 정통은 콘 토르티야 + 다양한 속. |
| **타말 (Tamal)** | 마사에 속을 넣어 옥수수 껍질 또는 바나나잎에 싸 찐 요리. 복수형 tamales. |
| **소페 (Sope)** | 두꺼운 마사 원판에 테두리를 만들어 콩·고기·살사를 올린 요리. |
| **토스타다 (Tostada)** | 오븐·기름에 바삭하게 구운 평평한 토르티야. 위에 콩·세비체·고기를 올린다. |
| **코말 (Comal)** | 토르티야 굽기·향신료 토스팅에 쓰는 납작한 멕시코 전통 철판. 무쇠팬·철그리들로 대체 가능. |
| **몰카헤테 (Molcajete)** | 화산석 절구. 살사·과카몰리를 가는 전통 도구. 블렌더로 대체 가능하나 질감이 다르다. |
| **토마티요 (Tomatillo)** | 살사 베르데의 핵심 재료. 초록 껍질(까데기)에 싸인 작고 신 과일. 토마토와 다른 식물. |
| **에파소테 (Epazote)** | 콩 요리·살사에 쓰는 멕시코 전통 허브. 독특한 쓴맛·허브 향. 호주에서 거의 없음. |
| **카르니타스 (Carnitas)** | 라드에 천천히 익혀 바삭하게 마무리한 돼지고기. 타코 필링의 대표. |
| **비리아 (Birria)** | 구아히요 베이스 소스에 조린 양·소·돼지고기 스튜. 비리아 타코(디핑 타코)로도 먹는다. |
| **포솔레 (Pozole)** | 호미니 옥수수와 고기를 끓인 수프. 포솔레 로호(붉은)·블랑코(흰)·베르데(초록). |
| **아치오테 (Achiote)** | 아나토(annatto) 씨앗으로 만든 붉은 마리네이드 페이스트. 코치니타 피빌의 핵심. |
| **크레마 (Crema)** | 멕시칸 사워크림. 산미와 크림이 균형 잡힌 드레싱용 유제품. 호주에서 사워크림+우유로 대체. |

## 마무리

이 부록이 책 전체의 출구가 되길 바란다. 레시피를 따라가다 막히는 순간, 장을 보다 명칭이 헷갈리는 순간, 재료가 없어 대체재를 찾는 순간 — 그때마다 여기로 돌아오면 된다.

멕시칸 요리는 완벽한 재료 목록이 갖춰졌을 때 비로소 시작하는 음식이 아니다. 있는 것으로, 지금 구할 수 있는 것으로, 먼저 해보는 것이 멕시코 주방의 정신이기도 하다. 브리즈번 부엌에서도 그 정신은 충분히 살아 있다.
