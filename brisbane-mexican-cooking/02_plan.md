# 호주 브리즈번 거주자를 위한 멕시칸 요리책 저술 계획

> 슬러그: `brisbane-mexican-cooking` · 장르: practical(요리책) · 활성 프로필: `profiles/practical/`
> 기반 레퍼런스: `brisbane-mexican-cooking/01_reference.md` (검색 시점 2026-05-25)

## 제목 후보

1. **《브리즈번 키친, 멕시코를 담다》** — 톤: 따뜻하고 정착민 친화적. 포지셔닝: "호주에 사는 한국인이 자기 부엌에서 정통 멕시칸을 해낸다"는 생활 밀착 메시지. 지역성(브리즈번)과 정통성(멕시코)을 한 제목에 묶어 타깃 독자가 즉시 자기 책으로 인지.
2. **《타코 너머: 정통 멕시칸 요리, 한국인의 손맛으로》** — 톤: 살짝 도전적·교양적. 포지셔닝: Tex-Mex/타코 키트로 알던 멕시칸의 "그 너머"를 정통으로 안내하되, 한국인 입맛 보정을 정체성으로 내세움. 멕시칸을 이미 조금 아는 중급 독자에게 강하게 어필.
3. **《마사에서 몰레까지: 호주에서 만드는 진짜 멕시코 요리》** — 톤: 전문적·체계적. 포지셔닝: 기본기(마사·닉스타말화)에서 정점(몰레)까지의 난이도 사다리를 제목으로 약속하는 "완주형" 요리책. 조달 가이드까지 포함한 호주 한정 실용서임을 명시.

**추천: 1번 《브리즈번 키친, 멕시코를 담다》**
이유: 타깃 독자(브리즈번 거주 한국인)가 제목만으로 "내 책"임을 인지한다. 정통성과 지역 조달이라는 이 책의 두 기둥을 모두 품으면서도 진입 장벽이 낮고 따뜻하다. 2번은 부제로 흡수(부제 "정통 멕시칸 요리, 한국인의 손맛으로" 활용 가능), 3번의 "마사에서 몰레까지"는 내러티브 아크 카피로 재활용한다.

## 책 특성

- **장르:** practical — 정통 기준 요리책 + 호주 현지 조달 가이드 + 한국인 입맛 보정서. 역사·문화 교양 요소 병합형.
- **분량:** 포괄적. 15개 챕터, 요리/핵심 레시피 **44종**(타깃 40종+ 충족). 챕터당 본문 9,000~16,000자, 레시피 항목당 1,500~4,000자(scaffolds 가이드 준수). 전체 한글 약 18만~21만 자 예상.
- **난이도:** 초급 진입 → 중급 완주. 1~5장은 도구·기초가 없는 완전 초보도 따라오게, 6장 이후 기법이 누적되며 중급 난이도(타말·몰레·비리아)로 상승. 각 레시피에 `::: meta` 난이도(하/중/상) 표기.
- **독자 여정(practical = "할 수 있게 되는 것"):**
  - **진입 상태:** Old El Paso 키트와 밀 토르티야 타코만 알고, 마사·토마티요·말린 고추가 뭔지 모르며, "호주에선 멕시칸 재료를 못 구한다"고 체념한 한국인 가정 요리사.
  - **출구 상태:** 브리즈번에서 마사 하리나와 말린 고추를 어디서 사는지 알고, 토르티야를 직접 눌러 굽고, 살사·과카몰리·타코·엔칠라다·비리아·몰레까지 정통 기준으로 만들며, 김치·고추장·된장으로 자기 입맛에 맞게 보정할 줄 알고, 손님상 멕시칸 메뉴를 한 상 차려낼 수 있는 사람.

## 내러티브 아크

이 책은 "마사에서 몰레까지" 난이도와 기법이 한 방향으로 누적되는 사다리로 설계했다. 요리책이라 각 레시피는 단독으로도 찾아 쓸 수 있어야(독립성) 하지만, 처음부터 순서대로 따라오면 초보가 중급 멕시칸 요리사로 자라도록(연속성) 흐름을 짠다.

**왜 이 순서인가.** 먼저 **왜 이 음식인가**를 납득시켜야 한다(1장: 역사·문화·UNESCO·정통 vs Tex-Mex, 그리고 한·멕의 의외의 공통점이라는 정서적 다리). 그다음 호주 독자의 가장 큰 현실 장벽인 **"재료를 못 구한다"는 체념을 2장에서 먼저 깨준다** — 조달이 해결돼야 나머지 레시피를 실제로 시도하기 때문이다. 3장에서 **도구와 기초 기법**(말린 고추 다루기·토스팅·블렌딩·콩 삶기)을 한 번에 정리해 이후 챕터가 이를 반복 참조하게 만든다.

본격 요리는 **쉬운 것부터 쌓는다.** 4장 살사·과카몰리(불 없이/최소 조리, 즉각 성취감) → 5장 토르티야(닉스타말화·마사·프레스, 모든 후속 요리의 토대) → 6장 타코(토르티야에 속을 채우는 가장 친숙한 형식, 카르니타스·알 파스토르·카르네 아사다·생선 타코) → 7장 엔칠라다·소페·토스타다(토르티야 응용 확장) → 8장 타말(마사를 찌는 한 단계 위 기법) 순으로 **토르티야/마사 계열을 한 흐름으로 묶었다.** 이어 9장에서 **지역의 정점 요리 몰레**(가장 복잡한 소스, 책의 클라이맥스)로 올라가고, 10장 수프·스튜(포솔레·비리아·코치니타 피빌 — 긴 조리·지역색), 11장 해산물(세비체·해안 요리), 12장 디저트·음료(플란·추로스·아구아 프레스카·마르가리타 — 식사를 마무리)로 밀도를 낮춘다.

마지막 두 장은 **이 책만의 약속을 회수한다.** 13장 한·멕 퓨전(김치 타코·고추장 아도보 — 앞에서 익힌 정통 위에 한국 손맛을 의도적으로 얹는 응용, "정통을 알아야 잘 비튼다"의 증명), 14장 메뉴 구성·플레이팅(배운 요리들을 한 상으로 조립하는 종합·플랜형), 15장 부록(조달처·명칭·대체재·도구·사진영상 레퍼런스 총정리, 두고두고 펼쳐보는 참조 챕터).

**역사·문화 배치:** 전용 1장(세계관)을 두되, 각 요리 챕터 도입부에 짧은 `::: tip` 또는 단락형 "이야기 박스"(그 요리의 유래·지역·문화 한 토막)를 얹어 교양을 본문 전체에 분산한다. 별도 챕터 + 도입부 박스의 이중 배치.

**호주 조달 배치:** 전용 2장(브리즈번 조달 가이드)을 앞에 두어 체념을 먼저 깨고, **추가로 모든 레시피의 재료 블록 끝에 "호주 조달 메모"**(어디서 사나·대체재·명칭 차이)를 `::: tip`으로 병기한다. 15장 부록에서 조달처를 한 번 더 표로 총정리. 전용 챕터 + 레시피별 메모 + 부록의 삼중 배치.

**한국인 보정 원칙(전 챕터 공통):** 레퍼런스 5장 원칙 준수 — **정통 레시피를 본문으로 먼저 완성한 뒤, 보정은 `::: tip` "한국인 입맛 보정" 콜아웃으로 선택지 병기.** 보정이 정통을 덮어쓰지 않는다. 13장만 예외적으로 퓨전을 본문 주제로 다루되 "정통성 주장이 아닌 제안"으로 프레이밍.

## 레시피별 공통 구성 규약 (전 요리 챕터 적용)

각 요리는 scaffolds의 "레시피·실습형" 스캐폴드를 따른다:

```
[도입] 무엇을 만드는가 + 한 줄 매력 + (이야기 박스: 유래·지역 한 토막)
::: meta  소요 시간 · 난이도 · 분량 · 필요 도구
::: ingredients  분량·단위 명기 (정통 재료)
  └ ::: tip 호주 조달 메모 (어디서·대체재·명칭 차이)
::: steps  번호 단계, 한 단계 한 동작, 이유 한 줄, 위험은 동작 앞에
[완성·확인] "이렇게 되면 성공"
[흔한 실수 / 복구]
::: tip 한국인 입맛 보정 (정통 위에 선택지로)
[응용·변형] 1~2가지 + **채식 버전 한 줄(규약)** — 고기를 콩·버섯·호박 등으로 바꾸는 길. 멕시칸의 3대 작물(옥수수·콩·호박)이 본래 채식 친화적이라는 강점 활용. 별도 챕터 없이 각 요리에서 규약화하고, 14장 메뉴 매트릭스의 채식 칸이 이들을 모은다
[그림: 사진/영상 참고 — 레퍼런스 7장 URL]
```

## 챕터 목록

### 1장. 멕시코를 먹는다는 것 — 옥수수·고추·발효의 5천 년
- **핵심 질문:** 멕시칸 요리는 왜 "타코"가 전부가 아니며, 한국인인 내게 왜 의외로 친숙한가?
- **주요 내용:**
  - **[도입 훅] 한·멕의 의외의 공통점**(고추 중심·발효·쌈/싸 먹기·곡물 주식+곁들임) — 책 전체 훅이므로 1장 **도입부**에 배치해 독자를 먼저 붙잡는다(기존 챕터 끝 → 앞으로 이동)
  - 메소아메리카 3대 작물(옥수수·콩·호박) + 고추·토마토·카카오 — 5천 년의 토대
  - UNESCO 인류무형문화유산(2010, 세계 최초 요리 분야 등재)과 미초아칸 패러다임·밀파·치남파
  - 스페인 정복(1521~) 이후 융합 — 돼지·소·치즈·라드가 만든 오늘의 멕시코 요리
  - 정통 멕시칸 vs Tex-Mex 명확히 구분(흰 치즈·콘 토르티야·처음부터 만드는 소스 vs 노란 체다·간 소고기·밀 토르티야) — 이 책의 기준선
  - 지역 지도: 오악사카(몰레의 땅)·유카탄(마야·코치니타)·푸에블라(몰레 포블라노)·북부(소고기·밀)
- **독자가 얻는 것:** "정통 멕시칸"이 무엇인지, 자기가 알던 Tex-Mex와 어떻게 다른지 분별하는 눈. 그리고 "낯설지만 통한다"는 친근감.
- **예상 분량:** 약 12,000자

### 2장. 브리즈번에서 멕시칸 재료 구하기 — "못 구한다"는 체념 깨기
- **핵심 질문:** 호주(브리즈번)에서 마사 하리나·말린 고추·토마티요·queso fresco를 정말 못 구하나? 못 구하면 무엇으로 대신하나?
- **역할 분리(중복 제거):** **2장은 "읽고 이해하는 내러티브 + 장보기 로드맵"** — 어디서·왜·어떻게 구하는지 서술 중심. 전체 조달처 디렉터리·명칭 대조표·대체재 종합표 같은 **참조용 표는 15장 부록으로 포워딩**("표로 찾아볼 땐 15장"). 2장은 서술, 15장은 표.
- **주요 내용:**
  - Woolworths/Coles/ALDI/IGA에서 바로 되는 것(아보카도·라임·고수·생고추·콩 통조림 등) vs 안 되는 것 — 서술형
  - 호주 식품 명칭 차이 핵심만 짚기(coriander=고수, capsicum=피망, eschalot/shallot 혼동 주의) — 전체 대조표는 15장. [확인필요 항목 "현지 1차 확인 권장" 고지]
  - 구하기 어려운 재료를 어떤 전략으로 구하나(슈퍼→라틴샵→온라인→대체재 순서의 의사결정 흐름) — 종합 대체재 표는 15장
  - 브리즈번/호주 조달처 소개: LatinShop(브리즈번 실물 매장)·La Tortilleria·El Cielo·Chile Mojo·Latin Deli·Amazon.com.au [매장 정보 변동 — 방문 전 확인 고지]
  - **한인마트·아시안 그로서리 활용**(고춧가루·청양고추·미림 + 라임·고수·생고추) — 보정과 대체의 보고
  - `::: tip` **상대 가격 체감 한 줄**(절대가 아닌 상대): "마사 하리나 2kg ≈ 슈퍼 밀가루의 몇 배" 식 감각, 변동 고지 유지
- **독자가 얻는 것:** 장보기 로드맵. "이건 슈퍼, 이건 LatinShop, 이건 한인마트, 정 없으면 이 대체재" — 나머지 챕터를 실제로 시도할 자신감.
- **예상 분량:** 약 13,000자

### 3장. 도구와 기초 기법 — 코말·프레스·블렌더, 그리고 고추 다루는 법
- **핵심 질문:** 멕시칸 요리를 시작하려면 무슨 도구가 필요하고, 모든 요리에 반복되는 기초 동작은 무엇인가?
- **주요 내용:**
  - 도구 4종(코말/무쇠팬·토르티야 프레스·몰카헤테·블렌더) + 호주 구매처·대체재 + **초보 진입 순서**(무쇠팬→블렌더→프레스→몰카헤테)
  - **말린 고추 다루기**(앙초·구아히요·파시야·치포틀레 비교표, 토스팅·불리기·씨 제거·블렌딩) — 살사·몰레·스튜의 공통 선행 기법
  - 코말 토스팅·차링(고추·토마토·양파·마늘) 기본기
  - **콩 삶기와 refried beans**(핀토/검정콩, 에파소테/오레가노) — 곁들임의 토대
  - 닉스타말화 원리 최소 설명(영양·펠라그라 — 5장 토르티야로 연결되는 다리)
  - `::: tip` 한국인 보정: 고춧가루로 색·향, 멕시칸 고추로 매움 깊이 — 매운맛 분리 제어
- **독자가 얻는 것:** 이후 모든 챕터가 전제하는 도구 세팅과 "고추 다루기·토스팅·콩 삶기" 3대 기초 기법.
- **예상 분량:** 약 13,000자

### 4장. 살사와 과카몰리 — 불 없이 시작하는 첫 성취
- **핵심 질문:** 가장 먼저, 가장 쉽게 "정통 맛"을 내려면 무엇부터 만들까?

### 5장. 토르티야 — 닉스타말화에서 직접 굽기까지, 그리고 남은 것 다루기
- **핵심 질문:** 정통 멕시칸의 심장, 콘 토르티야를 호주 부엌에서 직접 만들 수 있나? 한 번에 다 못 쓰고 남으면 어떻게 보관·재가열하나?
- **주요 내용 (레시피):** ① 마사 하리나로 콘 토르티야(프레스·코말) ② (심화) 마른 옥수수부터 닉스타말화 ③ 밀 토르티야(북부식, 라드/식용유) ④ 토르티야 칩·토스타다 셸
- **[독립 절 — 비중 격상] 남은 마사·토르티야 보관과 재가열** (한 항목이 아닌 별도 절로 격상 — 6·7·8장이 모두 5장 토르티야를 전제하므로 초보가 여기서 좌절하지 않게):
  - 남은 **마사 반죽** 보관(냉장 기간·마름 방지·되살리기)
  - 구운 **토르티야** 보관: 냉장 vs 냉동(겹침 분리·해동)
  - **재가열 3법 비교**: 코말(식감 복원 최선) vs 전자레인지(빠름·눅눅 주의) vs 오븐/포일(여러 장 한 번에) — 상황별 선택

  - "토르티야를 미리 만들어 보관해두면 6·7·8장이 수월하다"는 전제 명시
- **독자가 얻는 것:** 모든 타코·엔칠라다·토스타다의 토대인 토르티야 자급 능력 + 남은 것을 버리지 않고 다음 챕터로 넘어가는 자신감. 콘밀≠마사 하리나 분별.

### 9장. 몰레 — 수십 가지를 갈아 만드는 멕시코의 정점

  - `::: tip` 보정: 된장 소량으로 감칠맛 깊이(정통 위 선택)
- **독자가 얻는 것:** 책의 클라이맥스. 3장 고추 다루기가 정점에서 결실 맺는 경험. 평일판으로 성공한 뒤 정통판에 도전하는 양손잡이.

### 14장. 한 상 차리기 — 메뉴 구성과 플레이팅

### 15장. 부록 — 조달처·명칭·대체재·도구·레퍼런스 총정리
- **핵심 질문:** 두고두고 펼쳐볼 참조 정보를 어디에 모아두나?
- **역할 분리(중복 제거):** **15장은 "찾아보는 레퍼런스 표"** — 2장이 서술로 푼 내용을 표로 한곳에 모은다. 2장과 같은 표를 두 번 싣지 않고, 2장에서 "전체 표는 15장"으로 포워딩받는다.
- **주요 내용:**
  - 브리즈번/호주 조달처 디렉터리(매장·온라인, 변동 고지)
  - 호주↔멕시코↔한국 식품 명칭 대조표
  - 재료 대체재 종합표(마사·토마티요·치즈·고추·crema)
  - 말린 고추 비교표(재인용) + 도구 진입 순서

## 레퍼런스 한계 반영 (집필 단계 보강 필요)

레퍼런스 9장에 명시된 한계를 계획에 반영했다:
- **커뮤니티 페인포인트 얕음** → 각 챕터 "흔한 실수/복구" 박스는 집필 시 보강 필요(초보 실패담).
- **브리즈번 초로컬 정보 변동** → 2·15장 조달처는 "방문 전 확인" 고지 필수, 한인마트 구체 상호는 미수집(집필 시 일반 안내로).
- **호주 명칭 부분 확인** → capsicum/eschalot 등은 "현지 1차 확인 권장" 주석 유지.
- **퓨전 출처 신뢰도** → 13장은 "정통성 주장이 아닌 제안"으로 프레이밍(원칙 준수).
- **지역 요리 심화 부족** → 8·9·10장(타말·몰레·코치니타)은 추가 지역 전문 소스 리서치 권장.
- **이미지 라이선스 개별 미검증** → 15장 사진 링크는 파일별 라이선스 집필 시 확인.

> **보고:** 리서치 한계 1·5번(커뮤니티 실패담·지역 요리 심화)은 챕터 저술 단계에서 추가 리서치 보강이 필요할 수 있다.
