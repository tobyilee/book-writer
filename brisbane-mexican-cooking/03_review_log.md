## 레퍼런스 → 챕터 매핑 (대조표)

| 레퍼런스 섹션 | 매핑된 챕터 | 반영도 |
|---------------|-------------|--------|
| 1. 개념·정의 (3대 작물·닉스타말화·마사 vs 마사하리나·정통 vs Tex-Mex·타코) | 1장, 일부 3장(닉스타말화)·5장 | 충실 |
| 2. 역사·문화 (UNESCO·메소아메리카·정복·지역별·몰레·한식 공통점) | 1장 + 각 장 이야기 박스 | 충실 |
| 3.1 옥수수·토르티야 | 5장 | 충실 |
| 3.2 말린 고추 비교표 | 3장 + 15장 재인용 | 충실 |
| 3.3 살사·과카몰리 | 4장 | 충실 |
| 3.4 콩·허브·세비체·타말·소페·토스타다·비리아·카르니타스·마리네이드 | 3·6·7·8·10·11장 분산 | 충실 |
| 4.1 호주 슈퍼 조달 + 명칭 차이표 | 2장 | 충실 |
| 4.2 대체재 표 | 2장 + 15장 | 충실 |
| 4.3 브리즈번 조달처 | 2장 + 15장 | 충실 |
| 4.4 아시안/한인 그로서리 | 2장 + 보정 팁 전반 | 충실 |
| 5. 한국인 보정 원칙 | 전 장 `::: tip` + 13장 본문 | 충실 |
| 6. 도구 | 3장 + 15장 | 충실 |
| 7. 사진·영상 링크 | 각 장 + 15장 | 충실 |
| 9. 리서치 한계 | "레퍼런스 한계 반영" 섹션 | 명시적 반영 |

**대조 결론:** 레퍼런스 커버리지는 거의 빠짐없이 챕터로 흡수됐다. 레퍼런스에 있으나 계획에서 누락된 1차 콘텐츠는 없다. 문제는 "레퍼런스에 없지만 대상 독자가 반드시 필요로 하는 것"(보관·재가열, 식사 계획의 실전성, 가격 감각, 분량 환산)과 챕터 균형·프레이밍 일관성 쪽이다.

## 리뷰 라운드 1 (plan-reviewer → book-planner)

### Critical (반드시 반영)

- **[5장 ↔ 6·7·8장 전제 위반] 토르티야 보관·재가열이 5장 안에 묻혀 있어 독자 여정이 끊긴다.**
  - 근거: 누락 관점 + 흐름. 5장 ⑤에 "보관·재가열법"이 한 항목으로 들어가 있으나, 6·7·8장은 모두 5장에서 만든 토르티야를 전제로 한다. 초보는 토르티야를 한 번에 다 못 쓰고 남긴다 — 보관·재가열이 부실하면 6장으로 넘어가기 전에 좌절한다. 레퍼런스 자체에도 보관 항목이 없어(섹션 9 한계와 별개) 집필 시 채워야 할 공백이다.
  - 제안: 5장 ⑤를 독립 비중으로 격상하고, "남은 마사/반죽 보관", "구운 토르티야 냉장·냉동·재가열(코말 vs 전자레인지 vs 오븐)"을 명시. 14장(한 상 차리기)의 make-ahead와 교차 참조 거는 것을 계획에 박아둘 것.

- **[9장 몰레 진입장벽] 책의 클라이맥스가 초보 졸업 직전에 좌절 지점이 될 위험. 분량(18,000자)도 최대.**
  - 근거: 챕터 균형 + 독자 적합. 레퍼런스 섹션 2·3은 몰레를 "수십 가지 재료"로 규정. 대상 독자는 중급 진입 단계인데, 9장은 가장 길고(18k, 1장의 1.5배) 가장 어렵다. 평일 간소화판(②)을 넣은 것은 좋으나, 계획상 "정통 포블라노 → 간소화" 순서면 초보가 첫 페이지의 재료 목록 길이에서 이탈한다.
  - 제안: 9장 서술 순서를 "간소화 평일 몰레(②) 먼저 → 정통 포블라노(①)는 도전 과제로" 뒤집을 것. 또는 도입부에 "이 장은 주말 프로젝트다 / 재료는 15장 부록에서 한 번에 장보기"라는 기대치 관리 박스를 명시. 분량은 16k 선으로 조정 권장.

- **[프레이밍 일관성 — 4장 살사 베르데] "정통=본문 / 보정=팁" 원칙이 4장에서 깨진다.**
  - 근거: 독자 적합 + 프레이밍. 4장 ③ 살사 베르데가 본문에서 "토마티요 — 호주 대체: 그린 토마토+라임"으로 제목 줄에 대체재를 끌어올렸다. 이는 "정통을 본문, 조달 보정을 팁으로"라는 자기 원칙(내러티브 아크 36행, 레퍼런스 5장)과 충돌 — 본문이 대체재로 시작된다.
  - 제안: 본문은 토마티요 정통 레시피로 쓰고, 그린 토마토+라임 대체는 `::: tip` 호주 조달 메모로 내릴 것. 같은 점검을 7장 엔칠라다(queso fresco 대체), 10장(신 오렌지 대체)에도 일괄 적용 — 제목/본문 줄에 대체재가 올라온 곳을 전수 검사.

### Should (반영 권장)

- **[14장 식사 계획의 실전성] "메뉴 매트릭스 + 타임라인"이 추상에 그칠 위험. 남은 재료 활용·1인/2인 분량 환산이 빠졌다.**
  - 근거: 누락 + 독자 적합. 브리즈번 거주 한국인 다수는 소가구(1~2인)다. 레시피 분량은 보통 4~6인 멕시칸 기준이라, 남은 재료(마사·말린 고추·살사) 활용과 분량 축소가 실전 핵심인데 14장이 "손님상·파티" 위주로만 보인다.
  - 제안: 14장에 "남은 재료 리믹스(살사→엔칠라다 소스, 남은 카르니타스→케사디야)"와 "1~2인 분량 환산 가이드" 절을 추가. 평일 30분 코스를 소가구 기준으로.

- **[2장 ↔ 15장 중복] 조달처·명칭표·대체재표가 2장과 15장에 거의 동일하게 두 번 들어간다.**
  - 근거: 중복. 2장(조달 가이드)과 15장(부록)이 "조달처 디렉터리·명칭 대조표·대체재 종합표"를 모두 담는다. 인접하지 않아 치명적이진 않으나 18~22만 자 분량에서 동일 표 중복은 낭비다.
  - 제안: 2장은 "읽고 이해하는 내러티브 + 장보기 로드맵"으로, 15장은 "찾아보는 레퍼런스 표"로 역할을 명확히 분리. 같은 표를 두 번 싣지 말고 2장은 서술·15장은 표, 또는 2장에서 "전체 표는 15장"으로 포워딩.

- **[채식 옵션 산재] 채식 독자 동선이 14장 매트릭스 한 칸과 8장 채식 타말에만 흩어져 있다.**
  - 근거: 누락. 레퍼런스 3대 작물(옥수수·콩·호박)은 본질적으로 채식 친화적이다. 멕시칸은 채식 전환이 쉬운 요리인데 계획이 이 강점을 살리지 못함.
  - 제안: 별도 챕터까지는 불필요. 각 요리 챕터의 `[응용·변형]`에 "채식 버전" 한 줄을 규약으로 박고, 14장 매트릭스의 채식 칸이 그것들을 모으도록 교차 참조.

### Nice (여유 있으면)

- **[가격 감각 부재]** 레퍼런스 섹션 9-6이 가격 전무를 인정. "마사 하리나 2kg ≈ 슈퍼 밀가루 몇 배" 같은 상대적 체감(절대가 아님)을 2장에 한 줄 넣으면 조달 결심에 도움. 변동 고지 유지.

- **[11장 세비전 안전 비중]** 생식 안전 `::: warning`은 적절. 다만 호주 RAW 생선 표기(sashimi-grade 여부)·임산부 주의 한 줄을 더하면 신뢰도↑.

- **[1장 한·멕 공통점의 위치]** 훅으로 강력하나, 1장 끝(70행)에 배치돼 있다. 책 전체 훅이라면 1장 **도입부**로 끌어올려 독자를 먼저 붙잡는 편이 낫다(취향 영역이므로 저자 판단 존중).

### 전체 평
레퍼런스 흡수도와 "마사에서 몰레까지" 난이도 사다리 설계는 탄탄하다 — 2장(조달)으로 체념을 먼저 깨고, 5장 토르티야를 토대로 6·7·8장을 묶고, 13장에서 정통+보정 약속을 회수하는 아크는 practical 장르의 모범에 가깝다. 핵심 보완점은 세 가지로 압축된다: (1) 토르티야 보관·재가열을 일급으로 격상해 챕터 간 전제를 잇고, (2) 몰레의 진입장벽을 순서·기대치 관리로 낮추고, (3) "정통=본문/보정=팁" 프레이밍을 4·7·10장 본문 줄에서 누수 없이 지키는 것. 채식·소가구·남은 재료는 별도 챕터 없이 기존 구조 안에서 규약화하면 충분하다.

## 라운드 1 반영 (book-planner → 02_plan.md 갱신)

리뷰 라운드 1 피드백을 다음과 같이 반영했다. 변경 사유는 항목별 제안에 근거한다.

### Critical (전부 반영)
1. **[5장 토르티야 보관·재가열 격상]** ⑤ "보관·재가열법" 한 항목을 **"남은 마사·토르티야 보관과 재가열" 독립 절**로 격상. 남은 마사 보관 / 토르티야 냉장·냉동 / **재가열 3법 비교(코말 vs 전자레인지 vs 오븐)** / 굳은 토르티야 재활용 명시. **14장 "남은 재료 리믹스"와 교차 참조**, "미리 만들어두면 6·7·8장이 수월하다" 전제 명문화. 제목·핵심질문·예상분량(14k→15k) 조정. 사유: 6·7·8장이 5장을 전제하므로 보관 부실 시 초보 이탈 방지.
2. **[9장 몰레 진입장벽]** 레시피 서술 순서를 **"① 간소화 평일 몰레 → ② 핍피안 베르데 → ③ 몰레 포블라노(도전 과제)"**로 뒤집음(성공 경험부터). 도입부에 **기대치 관리 박스**(`::: tip` "이 장 읽는 법" — 주말 프로젝트·재료는 15장서 한 번에 장보기) 추가. 분량 18k→**16k** 조정. 사유: 클라이맥스가 좌절 지점이 되지 않게.
3. **[프레이밍 누수 전수 검사]** "정통=본문 / 보정·대체=`::: tip`" 원칙 누수 3곳 수정:
   - **4장 ③ 살사 베르데**: 제목 줄의 "호주 대체: 그린 토마토+라임" 제거 → 본문은 토마티요 정통, 대체는 조달 tip으로.
   - **7장 ① 엔칠라다**: queso fresco를 본문 정통으로, 파니르·페타 대체는 조달 tip으로.
   - **10장 ③ 코치니타 피빌**: 신 오렌지를 본문 정통으로, 라임+오렌지 대체는 조달 tip으로. 고추장 보강은 한국인 보정 tip으로 분리.

### Should (전부 반영)
- **[14장]** "남은 재료 리믹스"(살사→엔칠라다, 카르니타스→케사디야, 마사·토르티야→칩 — 5장과 교차) + "1~2인 소가구 분량 환산 가이드"(4~6인 기준 축소, 소량 어려운 재료는 만들어 냉동) 두 절 추가. 채식 칸이 각 챕터 채식 버전을 모으도록 명시.
- **[2장↔15장 중복]** 역할 분리 명문화 — **2장=서술/장보기 로드맵(표는 15장으로 포워딩)**, **15장=참조 표**. 동일 표 이중 게재 제거.
- **[채식 규약화]** 별도 챕터 없이 "레시피별 공통 구성 규약"의 `[응용·변형]`에 **채식 버전 한 줄 규약** 추가, 14장 매트릭스가 수집. 3대 작물(옥수수·콩·호박)의 채식 친화성 강점 명시.

### Nice (전부 반영)
- **[2장 가격 감각]** `::: tip` 상대 가격 체감 한 줄("마사 하리나 2kg ≈ 슈퍼 밀가루 몇 배") 추가, 변동 고지 유지.
- **[11장 세비체 안전]** RAW 생선 sashimi-grade 확인 + 임산부 주의 한 줄을 `::: warning`에 추가.
- **[1장 훅 위치]** 한·멕 공통점을 챕터 끝 → **도입부**로 이동(책 전체 훅).

### 부수 조정
- 책 특성 분량 요약: 챕터당 8,000~22,000자 → **9,000~16,000자**, 전체 18만~22만 → **18만~21만 자**(5장 +1k, 9장 −2k 반영).
